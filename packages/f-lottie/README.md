FEngine 的 [Lottie](https://airbnb.design/lottie/) 播放组件

See our GitHub [https://github.com/antvis/FEngine](https://github.com/antvis/FEngine) for more information

## Install

Using npm:

```sh
npm install --save @antv/f-lottie
```

or using yarn:

```sh
yarn add @antv/f-lottie
```
